# Projektkontext

Leitfrage: Ist der Wechsel auf das Berufliche Gymnasium, insbesondere SGG Soziales, für eine biologisch interessierte Schülerin fachlich ein Nachteil?

Verglichen werden:
- Basisfach Biologie
- Leistungsfach Biologie
- reguläre BG-Biologie
- Sondergebiete der Biowissenschaften als eigene optionale und schulabhängige Erweiterung

Nicht verwenden:
- Gesundheit und Biologie

Gesicherter Zwischenstand:
- Zentrale Themenfelder kommen in beiden Systemen vor.
- Das Leistungsfach ist stärker als vertieftes schriftliches Fach dokumentiert.
- Biologie ist im SGG Soziales nicht Profilfach.
- Das reguläre BG-Fach behandelt Populationsgenetik und Allelhäufigkeiten, nennt Hardy-Weinberg aber nicht ausdrücklich.
- Im Zusatzfach Sondergebiete der Biowissenschaften ist Hardy-Weinberg in BPE 6.4 ausdrücklich enthalten.
- Ob die konkrete Zielschule das Zusatzfach und die betreffenden Module anbietet, bleibt offen.
- Konkrete Unterrichtsqualität der Zielschule bleibt offen.

Die Daten zum Zusatzfach stehen getrennt in `data/sondergebiete.json` und werden nicht dem regulären BG-Pflichtcurriculum zugerechnet.
