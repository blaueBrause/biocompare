# Projektkontext

Leitfrage: Ist der Wechsel auf das Berufliche Gymnasium, insbesondere SGG Soziales, für eine biologisch interessierte Schülerin fachlich ein Nachteil?

Verglichen werden:
- Basisfach Biologie
- Leistungsfach Biologie
- reguläre BG-Biologie
- Sondergebiete nur optional

Nicht verwenden:
- Gesundheit und Biologie

Gesicherter Zwischenstand:
- Zentrale Themenfelder kommen in beiden Systemen vor.
- Das Leistungsfach ist stärker als vertieftes schriftliches Fach dokumentiert.
- Biologie ist im SGG Soziales nicht Profilfach.
- Konkrete Unterrichtsqualität der Zielschule bleibt offen.
