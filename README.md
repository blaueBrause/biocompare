# Biologie-Vergleich BW

Statische Entscheidungshilfe zur Schulwahl.

Leitfrage: Ist ein Wechsel auf das Berufliche Gymnasium, insbesondere SGG Soziales, für eine biologisch interessierte Schülerin fachlich ein Nachteil gegenüber dem Verbleib am allgemeinbildenden Gymnasium mit Biologie-Leistungsfach?

## Struktur

- `index.html` – self-contained HTML-Auswertung.
- `daten.json` – strukturierte Arbeitsdaten.
- `docs/` – gelernte Informationen, offene Fragen, Quellenliste.
- `sources/` – verfügbare Quell-PDFs und Begleitdateien.

## Arbeitsregeln

- Keine Aussage ohne Quellenbezug.
- Materialien, die nicht zum Zielfall SGG Soziales passen, werden nicht verwendet.
- Abkürzungen werden sparsam verwendet; zentrale Begriffe stehen im Glossar.
- Sondergebiete der Biowissenschaften werden als optional und schulabhängig markiert.
